## 适合python开发的agent有哪些？



### openclaw

1. github：[https://github.com/openclaw/openclaw](https://github.com/openclaw/openclaw)
2. 其依赖的agent框架：[https://github.com/badlogic/pi-mono](https://github.com/badlogic/pi-mono)
3. 编程语言：ts



### copaw
0. 是什么：可以理解为python开发的“openclaw”
1. github：[https://github.com/agentscope-ai/CoPaw](https://github.com/agentscope-ai/CoPaw)
2. 其依赖的agent框架：
    - agentscope:所有的agent实现 [https://github.com/agentscope-ai/agentscope](https://github.com/agentscope-ai/agentscope)
    - agentscope-runtime:agent的调度管理和部署 [https://github.com/agentscope-ai/agentscope-runtime](https://github.com/agentscope-ai/agentscope-runtime)
    - reme: 记忆框架 [https://github.com/agentscope-ai/ReMe](https://github.com/agentscope-ai/ReMe)

3. 编程语言： python
4. 优点： 阿里开源，代码通熟易懂。




### qwen-agent
1. github : [https://github.com/QwenLM/Qwen-Agent](https://github.com/QwenLM/Qwen-Agent)
2. 编程语言：python
3. 优点：阿里开源，代码通熟易懂。
